# Dar Al Ber Digital Humanitarian Platform — Executive Demo V2

This is a static frontend executive demo for a proposed modernization concept for Dar Al Ber Society's website.

## Files

```text
index.html
styles.css
app.js
README.md
assets/
  logo.png
```

## How to run locally

Open `index.html` directly in a browser.

## How to deploy on GitHub Pages

Upload all files to the root of the repository:

```text
index.html
styles.css
app.js
README.md
assets/logo.png
```

Then enable GitHub Pages for the repository.

## Demo limitations

- No real payment is processed.
- No personal data is stored.
- Donor portal, impact dashboard, tracking IDs, charts, and forms are simulated.
- Official Dar Al Ber information is used as factual grounding only.
- Terminology, slogan, governance model, trust center, waqf lifecycle, and donation-to-impact model are proposed concept language.
